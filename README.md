# 像素塔防小游戏

一个经典的塔防策略小游戏，采用纯 HTML + CSS + JavaScript 单文件实现，无需任何外部框架或依赖。

## 在线体验

直接在浏览器中打开 `index.html` 文件即可开始游戏。

```bash
# 或使用本地服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 游戏特色

### 像素卡通风格
- 拟人化可爱防御塔：弓箭手、炮兵、冰法师，均有大眼睛和腮红
- 圆滚滚的可爱敌人，带有表情差异
- 草地风格地图，点缀像素草丛和小花
- 像素风开始界面，含城堡、旗帜、丝带标题

### 随机路径系统
- 每次开始游戏或重新开始时，敌人路径都会随机生成
- 路径从地图左侧入口蜿蜒至右侧出口
- 确保路径合理可达，不贴边

### 三种防御塔

| 防御塔 | 造价 | 攻击力 | 特点 |
|--------|------|--------|------|
| 箭塔（弓箭手） | 50 金币 | 10 | 攻速快，单体攻击 |
| 炮塔（炮兵） | 100 金币 | 30 | 范围爆炸伤害 |
| 冰塔（冰法师） | 75 金币 | 5 | 减速敌人 50% |

### 敌人种类

| 敌人 | 生命值 | 速度 | 奖励 |
|------|--------|------|------|
| 基础怪 | 50 | 1.5 | 10 金币 |
| 快速怪 | 30 | 3.0 | 15 金币 |
| 坦克怪 | 200 | 0.8 | 25 金币 |

### 游戏系统
- **波次系统**：共 10 波敌人，难度递增
- **升级系统**：防御塔可升级 2 次，提升攻击力和射程
- **经济系统**：击杀敌人获得金币，用于建造和升级
- **生命系统**：20 点生命值，敌人逃脱扣除 1 点

## 操作说明

| 操作 | 说明 |
|------|------|
| 鼠标点击 | 选择防御塔类型，点击草地放置 |
| 点击已有塔 | 查看信息并升级 |
| ESC | 取消选择 |
| 空格键 | 暂停/继续游戏 |

## 技术实现

- **Canvas 2D 渲染**：所有图形使用 Canvas API 绘制
- **像素风格**：`image-rendering: pixelated` 保持像素清晰度
- **游戏循环**：`requestAnimationFrame` 实现流畅动画
- **单文件架构**：HTML、CSS、JavaScript 全部内联

## 开发历史

1. **基础版本**：实现核心塔防玩法（路径、塔、敌人、投射物）
2. **界面改造**：添加左右侧边栏、作战情报面板、深色像素风 UI
3. **外观优化**：

   - 草地风格地图
   - 随机路径生成
4. **开始界面**：像素风城堡标题界面，游戏状态管理
